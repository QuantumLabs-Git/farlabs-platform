Core Technology Stack

Backend Infrastructure

Python-based architecture leveraging modified Petals codebase with FastAPI for async operations, Redis + Celery for payment processing, and PostgreSQL/MongoDB for data management.

Frontend Technology

Next.js 14+ with TypeScript, Tailwind CSS + shadcn/ui components, wagmi + viem for BSC wallet connections, and Socket.io for real-time updates.

AWS Infrastructure

ECS Fargate containerization, ALB load balancing, RDS PostgreSQL, ElastiCache Redis, S3 model storage, CloudFront CDN, and Route53 DNS management.

Smart Contracts

Solidity on BSC with OpenZeppelin contracts for security, Chainlink oracles for reliability scoring, and automated payment distribution.

Reliability & Rating System

Metrics Tracked

  • Uptime percentage (30-day rolling average)
  • Response latency measurements
  • Inference accuracy (spot checks)
  • Token/second throughput
  • Customer satisfaction (optional feedback)

Rating Formula

Base Score = (Uptime × 0.4) + (Speed × 0.3) +
            (Accuracy × 0.2) + (Satisfaction × 0.1)

Bonus/Penalty = (Base Score - 0.8) × 50% // ±10% max
Final Payment = Base Payment × (1 + Bonus/Penalty)
