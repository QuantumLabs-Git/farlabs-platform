üöÄ Quick Setup Guide

1 Install Docker

Download and install Docker Desktop from docker.com

2 Pull Far Labs GPU Node Image 
docker pull farlabs/gpu-node:latest
3 Run the Node with Your Wallet 
docker run -d \
  --gpus all \
  --name farlabs-node \
  -e WALLET_ADDRESS=0xYourWalletAddress \
  -e NODE_NAME=MyGPUNode \
  -p 8080:8080 \
  farlabs/gpu-node:latest
4 Verify Node Status 
docker logs farlabs-node
# You should see: "Node connected to Far Labs network"
# Visit http://localhost:8080 to see your node dashboard
